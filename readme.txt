README for Contact Details project

- Install python3 using anaconda (also can install using pip)

- Preferred IDE is Spyder

- Runs both on MacOS and Windows

- Technical Dependencies

	> 	Tkinter

	>	DateTime

	> 	Tkcalender

	>	SQLite

- Installing required dependencies
	
	> 	condo install -c anaconda tk (also can install using pip)
	
	> 	conda install -c trentonoliphat datetime (also can install using pip)
	
	> 	pip install tkcalender 
	
	>	pip install sqlite
